Source Note

这家店更像什么

米其林一星与整体定位来自官网 about 和 OpenTable；Beef Wellington / Dover sole 属公开 listing 信号，不是官网明文展开的完整菜单。

Best For

适合什么局

  • 纪念日和庆祝餐
  • 想要老派仪式感
  • 重视酒单和服务
Skip If
  • 不想为场景和服务付溢价
  • 希望轻松随意
  • 对正式欧陆菜无兴趣
Ordering Bias

怎么点更合理

  • 优先按经典路线点：Beef Wellington、Dover sole 这类 house-signature 信号最强
  • 把预算留给一两道主菜和酒，不必把菜单铺太满
  • 如果只是想吃得好，不追求 ceremony，它未必是性价比最高的那家
Cautions
  • 正式感和价格门槛都高，临时起意去吃容易觉得过头
  • 官网菜单页文本不展开，部分 signature 判断来自 OpenTable 与公开菜单信号

简短结论

这是庆祝型餐厅，不是随便吃顿饭；值不值更多取决于你是否买单它的一星、服务仪式感和重酒单场景。

这里的 `best for / skip if / order bias` 是基于菜单、店型、公开 listing 与评价信号整理后的编辑判断，不等同于店方原始表述。
